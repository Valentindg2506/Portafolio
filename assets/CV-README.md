# Placeholder para CV

Coloca tu CV en formato PDF aquí con el nombre `cv.pdf`

El botón de descarga en el portafolio apunta a: `assets/cv.pdf`

## Recomendaciones para tu CV:

- **Formato:** PDF
- **Tamaño:** Máximo 2MB
- **Nombre:** cv.pdf
- **Contenido sugerido:**
  - Datos personales
  - Resumen profesional
  - Educación (DAM destacado)
  - Experiencia (proyectos, prácticas)
  - Habilidades técnicas
  - Idiomas
  - Enlaces (GitHub, LinkedIn, Portafolio)

## Herramientas para crear CV:

- Canva
- Overleaf (LaTeX)
- Microsoft Word
- Google Docs
- Europass (formato europeo)

## Tip:
Mantén una versión actualizada y asegúrate de que coincida con la información en tu portafolio.
