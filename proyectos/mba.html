🎉 MBA - RSVP Evento

Aplicación web para gestionar confirmaciones de asistencia a eventos

Descripción General

MBA RSVP Evento es una aplicación web en PHP pensada para organizar eventos y gestionar la confirmación de asistencia de los invitados de manera sencilla y eficiente.

El panel de administración permite crear eventos, generar enlaces públicos únicos por token y consultar en tiempo real quién confirmó asistencia, qué producto trae cada persona y el estado de pago de cada invitado. También incluye recordatorios por WhatsApp y una vista optimizada para móvil.

Tecnologías Utilizadas


PHP 8.0+

MySQL

HTML5

CSS3

Tokens únicos

Flujo de la Aplicación

1
El administrador crea un evento (nombre, fecha, descripción) desde el panel de admin.
2
El sistema genera un enlace público con token único que el admin puede copiar y compartir.
3
Los invitados acceden al enlace y confirman su asistencia indicando qué producto llevarán.
4
El admin gestiona la lista de asistentes, estados de pago y envía recordatorios por WhatsApp.

Características Principales

  • Gestión de Eventos: Panel admin para crear eventos con nombre, fecha y descripción
  • Token Único por Evento: Enlace público seguro generado automáticamente para cada evento
  • Confirmación de Asistencia: Formulario público para que los invitados confirmen asistencia (Sí / No)
  • Lista de Compra Automática: Reparto automático de productos por asistentes confirmados
  • Control de Pagos: Seguimiento del estado de pago (pagado / pendiente) por asistente
  • Recordatorio WhatsApp: Botón para enviar mensaje preformateado directamente por WhatsApp
  • Vista Separada de Asistentes: Sección diferenciada entre quienes van (Sí) y quienes no (No)
  • Sistema de Roles: Admin (edita todo) y Coorganizador (solo lectura)
  • Diseño Mobile-First: Interfaz optimizada para uso desde el teléfono
