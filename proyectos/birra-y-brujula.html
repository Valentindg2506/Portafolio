Birra y Brujula

Web temática y navegación por secciones para planes y juntadas

Descripción General

Birra y Brujula es una web temática pensada para centralizar información de eventos y juntadas, con distintas vistas para actividades, votaciones, transporte y compra.

El proyecto prioriza una experiencia clara de navegación entre páginas, identidad visual coherente y una base simple para seguir creciendo en funcionalidades.

Tecnologías Utilizadas


HTML5

CSS3

JavaScript

PWA

Características Clave

  • Múltiples vistas: Páginas dedicadas para actividades, transporte, compras y votaciones
  • Navegación simple: Estructura clara para moverse rápidamente por todo el sitio
  • Identidad visual: Estilo unificado para dar personalidad al proyecto
  • Base escalable: Organización preparada para ampliar módulos y dinámicas sociales

Enfoque y Valor del Proyecto

El proyecto se planteo como una app web multipagina para organizar viajes en grupo de punta a punta: asistentes, votaciones, transporte, compras, actividades, itinerario y liquidacion final.

  • Flujo completo de viaje: Dashboard + modulos separados para cada etapa de planificacion
  • Persistencia real: Datos guardados en localStorage con claves de dominio (trip_*)
  • Uso sin conexion: Service Worker en sw.js para cache del shell y experiencia offline basica
