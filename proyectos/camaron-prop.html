Camaron Prop

Portal inmobiliario con búsqueda avanzada y panel de administración

Descripción General

Plataforma web inmobiliaria que permite a usuarios buscar viviendas por localidad y rango de precio, y a administradores gestionar el catálogo de propiedades desde un panel dedicado.

El proyecto cubre nueve municipios del área de Valencia e implementa un sistema de filtrado dinámico con consultas SQL parametrizadas. Se desarrolló en 4 iteraciones, añadiendo complejidad progresiva hasta un panel de administración funcional.

Tecnologías Utilizadas


PHP 8

MySQL

HTML5

CSS3

Características Clave

  • Búsqueda por localidad: Valencia, Alboraya, Torrent, Gandía, Sagunto, Paterna, Burjassot, Xàtiva, Cullera
  • Filtro de precio: Rango mínimo y máximo configurable por el usuario
  • Tarjetas de propiedades: 11 campos por vivienda (m², baños, habitaciones, estado, tipo…)
  • Panel de administración: Navegación lateral con secciones (Viviendas, Imágenes, Usuarios, Propietarios)
  • Alta de propiedades: Formulario completo con inserción en MySQL
  • Visualización de registros: Tabla HTML con todos los registros de la BD

Evolución del Proyecto

  • V1 — Inicio: Estructura básica y conexión a base de datos
  • V2 — Proceso formulario: Procesamiento de búsquedas con POST
  • V3 — Segundo filtro: Filtro combinado localidad + rango de precio
  • V4 — Admin Panel: Panel de administración completo con sidebar y CRUD
